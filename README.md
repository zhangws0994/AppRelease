# AppRelease
国内andriod应用市场一键发布， 豌豆荚，华为，小米。。。等等

1.整合国内安卓应用市场的所有场景发布。提供相关行业信息，关键描述等，自动生产各个发布渠道的描述信息。解决运营人员花费大量的时间编写文案还审核不通过的情况（各个渠道对应的文字限制的要求）
2.一建发布所需上架渠道，如审核被拒，邮寄，短信主动通知，让运营人员及时知道审核情况。提高发不的效率
3.审核被拒后，如文案描述等被拒，自动修护并重新上架

应用宝，华为，小米，豌豆荚
